# Herts Vapes mobile upload version

This version has NO folders so it is easier to upload from iPhone.

Upload every file in this folder to your GitHub repository root:

- index.html
- style.css
- script.js
- herts-vapes-logo.png
- hv-icon.png
- all product .png images
- README.md

Do not upload the zip file itself.

Then enable GitHub Pages: Settings > Pages > Deploy from branch > main > /(root).
